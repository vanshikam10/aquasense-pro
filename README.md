# AquaSense Pro — Smart Irrigation Dashboard

Live Demo: https://vanshikam10.github.io/aquasense-pro/aquasense_pro.html

Smart irrigation decision system for Indian crops using 8-signal prediction engine.

## Features
- FAO-56 Hargreaves-Samani ETo calculation
- 17 Indian crops with agronomic thresholds
- Soil moisture, EC, temperature signal fusion
- Irrigation volume recommendation in kL

## Tech
HTML · CSS · JavaScript · FAO-56 Agronomy
